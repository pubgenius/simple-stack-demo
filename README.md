## Project setup
Set up nextjs project w/ TypeScript and Tailwind
`yarn create next-app --typescript --with-tailwindcss with-tailwindcss-app`
https://github.com/vercel/next.js/tree/4d4f3093019179b1928ec07c16f38882241c0375/examples/with-tailwindcss
According to Tailwind's documentation https://tailwindcss.com/docs/guides/nextjs

`yarn add -D tailwindcss postcss autoprefixer`
`npx tailwindcss init -p`
